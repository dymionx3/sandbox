# M.V. Electronix LLC - HTML Course

This repository contains an interactive HTML course built with React, Vite, React Router, TypeScript, Tailwind CSS, and shadcn/ui components.

## Getting Started

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Open http://localhost:8080 to view locally.

## Deployment on Cloudflare Pages

1. Push your code to a Git repository (GitHub, GitLab, or Bitbucket).
2. In Cloudflare dashboard, create a new Pages project and connect your repo.
3. Select **Framework preset**: *React (Vite)*.
4. Set **Build command** to:
   ```
   npm run build
   ```
5. Set **Build output directory** to:
   ```
   dist
   ```
6. Click **Save and Deploy**. Your site will be live on the provided Pages domain.

Enjoy deploying your interactive HTML course!
